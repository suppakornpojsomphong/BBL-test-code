# Appointment Booking System
A simple web application built with FastAPI for managing appointment bookings. This project demonstrates user authentication, role-based access control (Admin/User), and basic data management.

## Setup & Installation
Clone or Extract the project folder
```bash
cd appointment_app
```


## Install Dependencies

```bash
pip install fastapi uvicorn python-multipart PyJWT jinja2
```
##  How to Run
1. Start the application using Uvicorn:
```bash
uvicorn main:app --reload
```
2. Once the server is running, open your browser and go to:
```bash
http://127.0.0.1:8000
```
